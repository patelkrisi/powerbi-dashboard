# Power BI Dashboard Project 
This repository contains a PowerBi dashboard created as a part of my data analytics learning journey.
The goal is to showcase skills in data cleaning, modeling, DAX calculations, and dashboard design.

## Project Structure 
- `data/` - Sample dataset used in the dashboard
- `media/` - screenshots and visual assets
- `queries/` - Power Query (M) scripts
-`measures/` - DAX measures 
- `README_backup/` - backup folder
